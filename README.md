# Guess My Number App

## React Native App

### Features:

- React Native.
- Hooks.
- Expo Custom Fonts.
- Expo Linear Gradient.
- FlatList.
- Custom Components (e.g Title, PrimaryButton, NumberContainer, etc..).
- Support different screen sizes, modes and different platforms.
